## Getting Started

Before using the website, ensure that it is hosted on a server or accessible via localhost. You can access it by visiting the URL provided by the hosting server or by running it on your local machine.

---

## User Features

### Register
1. Navigate to the Register page.
2. Fill in your details: username, email, and password.
3. Click **Submit** to create an account.
4. You will be redirected to the Login page upon successful registration.

### Login
1. Navigate to the Login page.
2. Enter your email and password.
3. Click **Login** to access your account.

### Homepage
1. After logging in, you will be directed to the Homepage, which displays featured products and website announcements.
2. Use the navigation menu to explore other sections.

### Shop
1. Navigate to the Shop section.
2. Browse products, and click on a product to view its details.
3. Select a size (if applicable) and click **Add to Cart**.

### Cart
1. Access the Cart page to view items added.
2. Increase or decrease the quantity of items directly in the cart.
3. Click **Checkout** when ready to proceed.

### Checkout
1. In the Checkout section, add or confirm your billing information.
2. Review your order and click **Place Order** to complete your purchase.

### Contact Us
1. Go to the Contact Us page.
2. Fill in your message and contact details.
3. Click **Send Message** to submit your inquiry.

## Admin Features

### Admin Login
1. Admins log in using the same Login page as regular users.
2. Use admin credentials to access the Admin Dashboard.

### Admin Dashboard
1. **Income Report**
   - View income reports filtered by date range using the date picker (Start Date and End Date).
2. **Manage Users**
   - View, edit, or delete registered users.
3. **Add Product**
   - Enter product details: name, price, stock, and image.
   - Click **Add Product** to save it to the inventory.
4. **View Clothes**
   - Edit or delete existing products directly from the inventory.
5. **New Admin**
   - Create a new admin by entering their name, email, phone, and password.
6. **Messages**
   - View messages sent by users through the Contact Us form.

## How to Run the Website

### Hosted Version
1. Visit the website using the URL provided by the hosting provider.
2. Proceed with registration, login, and other features as described above.

### Localhost Version
1. Clone the website repository or download the source code.
2. Install required dependencies by running:
   ```bash
   npm install
   ```
   or the equivalent command for your technology stack.
3. Start the server:
   ```bash
   npm start
   ```
   or run the server-specific start command.
4. Open a web browser and visit `http://localhost:3000` (or the specified port).

Feel free to reach out via the Contact Us page for support or feedback. Enjoy your experience!
